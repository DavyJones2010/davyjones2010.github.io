asciinema使用笔记

发表于 2022-06-11 更新于 2022-11-05 本文字数： 270 阅读时长 ≈ 1 分钟

安装软件

  • asciinema
  • svg-term-cli
brew install nodejs
npm install -g svg-term-cli

生成录屏文件.cast

asciinema rec a.cast

Ctrl + D 结束录屏

本地播放录屏文件

asciinema play a.cast

.cast文件转成.svg文件

需要先安装 svg-term-cli

cat a.cast | svg-term --out a.svg --term iterm2 --profile mymatrix

配置文件

$HOME/.config/asciinema/config

# mac # tools # asciinema # svg
