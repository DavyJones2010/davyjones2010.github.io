# Intel CPU Suffix

- Y: M-5Y71 Mobile extremely low power
- U: 6550U, Mobile power efficient 超低压处理器, for 移动端
- P: Performance optimized for thin and light laptops
- H: 9750H, High performance optimized for mobile
- K: 8700K, Unlocked
- Q: quad core

- refs: [Understanding Intel® Core™ Processor Number](https://www.intel.com/content/www/us/en/support/articles/000058567/processors/intel-core-processors.html)
